Examples of logic gate
solved exercises

Not sure how to display a Boolean function using logic gates? Use the examples down bellow for a quick remainder.

NAND gate

1.

Using NAND gates, realize minimum circuit for a function given with PDNO: f (x1, x2, x3, x4) = V (R0, R1, 2, 3, 5, 7, R8, 11, R12, 13).

y = x1&x2 V x3&x4 V x2&x4

y = x1&x2 V x3&x4 V x2&x4

Using De Morgan

x1Vx2

x1&x2

y = x1&x2 & x3&x4 & x2&x4

NOR gate

2.

Using NOR gates, realize minimum circuit for a function given with PKNO: f (x1, x2, x3, x4) = & (R4, R5, R6, 7, 10, 11, 12, 13, 14, 15).

y = x1&x3 V x2

y = x1&x3 V x2

Using De Morgan

x1Vx2

x1&x2

y = x1Vx3 V x2

y = x1Vx3 V x2

Multiplexer

3.

Realize function given with PDNO: f (x1, x2, x3) = V (0, 2, 3, 4) using multiplexer with two address inputs (m=2).

Demultiplexer

4.

Realize function given with PDNO: f (x1, x2, x3) = V (2, 3, 4, 5) using demultiplexer with two address inputs (m=2).
